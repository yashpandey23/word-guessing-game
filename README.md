
A simple Python word guessing game where players try to guess hidden words by entering letters one by one.

## Features
- Random word selection from a predefined list
- Letter-by-letter guessing
- Limited attempts (only wrong guesses count)
- Real-time progress display
- Win/lose conditions

## How to Play
1. Run the Python script
2. Guess letters one at a time
3. Correct guesses reveal letter positions
4. Wrong guesses decrease attempts
5. Guess the complete word before attempts run out!

## Technologies Used
- Python
- Random module
